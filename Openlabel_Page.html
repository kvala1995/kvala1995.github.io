Open Label

OpenLabel is an open source chrome extension that acts as a Google News Feed and Twitter reader that helps users identify mesia misinformation. It will generate a label upon interacting with a given news article.Before you begin using and/or modifying the extension, there are a few topics to go over.

How the extension works

Open Label informs the user over several characteristics from an article. These characteristics are shown below

  • Emotion
  • Creditability
  • Publisher
  • MBFC Media Bias
  • Viral Terms

Emotion is gathered through reading the articles text and using sentiment analysis. Positive words in the text generate positive values and negative words in the text generate negative values towards the overall emotion score. Creditability, Publisher, and MBFC Media Bias are all sourced from Media Bias/Fact Check. Viral Terms are found using the Google Trends API.

The extension analyzes an article when the user hovers the URL for said article (the option also exists to right click the link to activate the analysis). After the user has then hovered over a link our extension will begin to analyze the article. It will indicate to the user that it is loading to communicate the URL is being analyzed.
